Herzlich willkommen auf dem Nutanix Virtual Solutions Workshop: Citrix und Nutanix AHV
--------------------------------------------------------------------------------------

.. figure:: images/Intro.png

Herzlich Willkommen beim End User Computing Workshop mit Citrix Apps & Desktops. 

Dieser Solutions Workshop soll Ihnen aus erster Hand zeigen, warum Nutanix die ideale Plattform für Citrix Workloads darstellt. 
 

Was Sie in dieser Online-Session erwartet?
++++++++++++++++++++++++++++++++++++++++++

Neben den vielen Vorteilen einer Nutanix HCI Umgebung für virtuelle Desktops, wie lineare Skalierbarkeit bei konsistenter Performance, bietet Ihnen Nutanix zusätzliche Vorteile für den Betrieb Ihrer Citrix Umgebung, welche Sie in diesem Workshop kennen lernen werden:

- Integrierte Tools zur Migrating existierender Images von ESXi 
- Citrix Integration mit AHV
- Schnelle Bereitstellung von Desktops 
- Native File Services mit Nutanix Files 
- Native Mikrosegmentierung mit Nutanix Flow 
- Monitoring und Automatisierung mir Prism Ops


**Agenda: 09:30 - 11:00 Uhr Virtueller Nutanix Solutions Workshop**
 

Danach erhalten Sie einen persönlichen, zeitlich limitierten Demo-Zugang für praktische Übungen.
